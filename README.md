# Microservicio de Puntuación y Comentarios

Este repositorio es un microservicio para realizar una puntuacion y un comentario sobre un producto en un ecommerce para la materia de Arquitectura de Microservicios de la carrera de Ingeniería en Sistemas de la Universidad Tecnológica Nacional, Facultad Regional de Mendoza.

Se integra con los microservicios de la materia relacionados con el sistema ecommerce propuesto por la cátedra.
